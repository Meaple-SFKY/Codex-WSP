# Global AI Agent Rules

## Engineering Principles

- Prefer production-grade architecture.
- Keep UI modern, fluid, and minimal.
- Favor maintainability over hacks.
- Prefer reusable shared packages.
- All apps should support future AI-assisted expansion.

## Monorepo Rules

- Applications belong under `apps/`.
- Shared logic belongs under `packages/`.
- Prompts belong under `prompts/`.

## Codex Rules

- Always read `SPEC.md` before implementation.
- Keep `TASKS.md` updated.
- Preserve architecture consistency.
- Prefer incremental commits.
