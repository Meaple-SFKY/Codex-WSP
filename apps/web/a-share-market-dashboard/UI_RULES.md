# UI Rules

## Layout

- Keep charts visible at all times.
- Avoid modal-heavy interaction.
- Keep navigation persistent.
- Prioritize market data readability.

## Watchlists

- Dense but readable rows.
- Fast sorting.
- Sticky headers.

## Charts

- Support crosshair.
- Support zoom.
- Support pan.
- Support synchronized indicators.

## Color Rules

- Red for rise.
- Green for fall.
- Neutral gray for inactive states.

## Performance

- Avoid unnecessary rerenders.
- Prefer virtualization for long lists.
