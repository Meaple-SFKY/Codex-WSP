# Web Reader Agent Rules

## Product Vision

Create a premium online reading experience inspired by Apple Books.

## UX Requirements

- Minimalistic design
- Smooth transitions
- Glassmorphism where appropriate
- Responsive layouts
- Beautiful typography
- Comfortable reading mode

## Functional Requirements

- Search books online
- Import TXT/EPUB/PDF in future
- Local bookshelf
- Reading progress persistence
- Dark/light mode

## Engineering Requirements

- Production-grade code quality
- Modular architecture
- Future sync-ready data layer
- Future plugin architecture support
