# Product Specification

## Core Features

1. Beautiful Apple Books inspired UI
2. Online search
3. Import local TXT/EPUB books
4. Local bookshelf
5. Reading progress tracking
6. Smooth page transitions
7. Reader customization

## Future Features

- Cloud sync
- AI summarization
- AI translation
- Notes and highlights
- Multi-device sync
- Audiobook support

## Technical Direction

- Next.js App Router
- TypeScript-first
- Component-driven architecture
- Offline-ready foundation
