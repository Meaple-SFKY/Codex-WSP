# Design System

## Visual Direction

Inspired by Apple Books:

- Calm
- Elegant
- Typography-focused
- Spacious
- Minimalistic
- Fluid animations

## UI Principles

- Avoid clutter.
- Prefer whitespace.
- Use soft shadows.
- Use layered depth subtly.
- Typography is primary.
- Reading comfort is more important than density.

## Typography

- Large elegant section titles.
- Comfortable line height.
- Book-centric layout.
- Smooth scaling across devices.

## Animations

- Smooth transitions.
- Avoid excessive motion.
- Use spring animations.
- Reader interactions should feel premium.

## Color Modes

- Light
- Dark
- Sepia

## Bookshelf Style

- Large cover cards.
- Soft hover effects.
- Depth layering.
- Rounded corners.
