# Apple Books Web Reader

一个基于 Web 的高质量在线阅读器，设计风格参考 Apple Books。

## 目标

- 美观、现代化、Apple 风格 UI
- 在线搜索图书
- 导入本地 TXT / EPUB
- 流畅阅读体验
- 后续支持 AI 能力扩展

## 技术栈

- Next.js
- TypeScript
- TailwindCSS
- shadcn/ui
- Zustand
- React Query

## 启动方式

```bash
pnpm install
codex exec
```
