# UI Rules

## Layout

- Prefer centered layouts.
- Use large margins.
- Avoid dense interfaces.

## Components

- Rounded corners.
- Smooth hover states.
- Consistent spacing.
- Consistent typography.

## Reader Page

- Reader must feel distraction-free.
- Reader controls should auto-hide.
- Smooth page transitions.
- Comfortable reading width.

## Bookshelf

- Grid layout.
- Premium cover presentation.
- Lazy loading.
- Smooth scrolling.
