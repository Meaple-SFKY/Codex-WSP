# Reader Experience

## Philosophy

The reader should feel calm, immersive, and premium.

## Reading Comfort

- Comfortable typography.
- Adjustable font size.
- Adjustable spacing.
- Theme customization.

## Interaction

- Smooth page transitions.
- Natural scrolling.
- Minimal distractions.
- Instant response.

## Performance

- Fast startup.
- Low memory usage.
- Lazy loading large books.
