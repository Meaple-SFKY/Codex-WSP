# Apple Books Web Reader

A beautiful web-based reading application inspired by Apple Books.

## Goals

- Elegant Apple-like UI
- Online search
- Local book import
- EPUB/TXT support
- Smooth reading experience
- Future cloud sync support

## Stack

- Next.js
- TypeScript
- TailwindCSS
- shadcn/ui
- Zustand
- React Query

## Start

```bash
pnpm install
codex exec
```
