# SwiftUI Reader Agent Rules

## Product Vision

Create a premium native reading experience inspired by Apple Books.

## Requirements

- Native SwiftUI animations
- Smooth page transitions
- Native typography
- Reading comfort first
- Support iPhone, iPad, macOS

## Features

- Online search
- EPUB/TXT import
- Local bookshelf
- Reading progress
- Dark mode

## Engineering

- MVVM architecture
- Modular Swift packages
- Future sync-ready
