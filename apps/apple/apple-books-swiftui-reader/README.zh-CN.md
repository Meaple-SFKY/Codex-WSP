# Apple Books SwiftUI Reader

一个基于 SwiftUI 的 iOS/macOS 原生阅读器，设计风格参考 Apple Books。

## 目标

- 原生 SwiftUI UI
- 支持 iPhone / iPad / macOS
- 本地 EPUB / TXT 导入
- 在线搜索
- 流畅翻页与阅读体验

## 技术栈

- SwiftUI
- SwiftData
- Combine
- EPUBKit（或后续自定义解析器）

## 使用方式

使用 Xcode 打开工程后执行：

```bash
codex exec
```
