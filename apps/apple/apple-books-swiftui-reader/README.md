# Apple Books SwiftUI Reader

A native iOS/macOS reading experience inspired by Apple Books.

## Goals

- Native SwiftUI design
- iOS + macOS support
- Beautiful bookshelf UI
- Local EPUB/TXT import
- Online search

## Stack

- SwiftUI
- SwiftData
- EPUBKit / custom parser
- Combine

## Open

Open with Xcode and run:

```bash
codex exec
```
