# Tasks

## Phase 1: Native Foundation

- Create an Xcode-openable SwiftUI app structure.
- Support iOS, iPadOS, and macOS targets where practical.
- Build the main bookshelf screen inspired by Apple Books.
- Build the reading screen with beautiful typography and reading controls.
- Add mock book data for visual development.

## Phase 2: Local Import

- Implement TXT import.
- Implement EPUB import architecture.
- Parse metadata such as title, author, cover, and table of contents.
- Persist imported books locally.

## Phase 3: Online Search

- Implement online book search service abstraction.
- Start with Open Library or another public book metadata source.
- Display search results with cover, title, author, and source.
- Support importing/search-result bookmarking.

## Phase 4: Reader Experience

- Persist reading progress.
- Add font size, line spacing, theme, and page style controls.
- Add smooth page transitions.
- Add dark mode and sepia mode.

## Phase 5: Future Expansion

- Add notes and highlights.
- Add iCloud sync architecture.
- Add AI summary and translation entry points.
- Add audiobook-ready architecture.
