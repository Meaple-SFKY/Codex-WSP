# Product Specification

## Core Features

1. Apple Books inspired UI
2. Native SwiftUI bookshelf
3. EPUB/TXT import
4. Online search
5. Reading progress sync-ready architecture
6. Smooth page animation

## Future Features

- AI summaries
- Notes/highlights
- iCloud sync
- Audiobook mode
- Vision Pro support

## Technical Direction

- SwiftUI-first
- SwiftData persistence
- MVVM architecture
- Modular packages
